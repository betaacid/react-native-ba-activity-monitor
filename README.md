# react-native-ba-activity-monitor

Native Activity Monitor for RN apps

## Installation

```sh
yarn add react-native-ba-activity-monitor
```

## Usage

```js
import { multiply } from 'react-native-ba-activity-monitor';

// ...

const result = await multiply(3, 7);
```
